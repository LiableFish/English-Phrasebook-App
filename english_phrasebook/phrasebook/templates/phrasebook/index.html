Welcome to the English Phrasebook app developed with Django

About app:

App contains admin page for dealing with Category, Level, Theme and Word models. It also accepts some API requests with "Secret" header permission (for DEBUG=False).

There are some urls you can use:

  • Link to the admin panel
  • GET request to see categories list
  • GET request to see levels list
  • GET request to see themes list (it also accept query params "category" and "level")
  • GET request to see theme detail
  • GET request to see word detail
